# Personal Expense Tracker (Python GUI)

This is a simple desktop GUI app for tracking personal expenses.

## How to Run

1. Install Python 3.10+
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python gui_expense_tracker.py

## Features
- Add expense by category
- View and analyze expenses
- Reset all expenses
- Graphical bar chart inside GUI
